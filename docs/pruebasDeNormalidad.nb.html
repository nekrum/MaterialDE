Pruebas de Normalidad

Resumen

Si quieres saber si tus datos tienen una distribucin normal usa mtodos grficos y aplica pruebas estadsticas de normalidad:

Mtodos grficos * Histogramas: para ver la forma de la distribucin

datos = rnorm(100)
hist(datos)

  • QQ-plot: para ver cmo los cuantiles de los datos se ajustan a los cuantiles de la distribucin normal terica. Si los datos tienen una distribucin normal, los puntos tienen que seguir la lnea recta.
qqnorm(datos)
qqline(datos, col="red", lw=2)

Pruebas de normalidad * Hiptesis nula = datos tienen una distribucin normal * Con p-value < 0.05 se rechaza hiptesi nula * Shapiro-Wilk * para tamao de muestra menor a 5 mil

shapiro.test(datos)

    Shapiro-Wilk normality test

data:  datos
W = 0.99048, p-value = 0.7037
  • Kolmogorov-Smirnov: para tamao de muestra grande
    • se tiene que especificar la distribucin
    • Sirve para comparar contra distribuciones continuas
set.seed(100)
binom_data = rbinom(15, 5, .6)
shapiro.test(binom_data)

    Shapiro-Wilk normality test

Antes de analizar los datos con pruebas estadsticas paramtricas que asumen una distribucin normal, cmo pruebas T o ANOVAs, debemos asegurarnos que nuestros datos tienen una distribucin normal. Para ello podemos usar una combinacin entre pruebas estadsticas de normalidad y mtodos grficos como histogramas y qq-plots.

Pruebas estadsticas de normalidad

Las pruebas de normalidad son pruebas estadsticas que ponen a prueba la hiptesis nula (h0) de que los datos tienen una distribucin normal. Esto quiere decir que si la prueba arroja un valor de p < 0.05 (o el valor de significancia que decidas), se rachaza la h0 y concluimos que los datos NO estn distribuidos de manera normal.

Algunas pruebas son:

  • Shapiro-Wilk: usar slo para un tamao de muestra menor a 5 mil
  • Kolmogorov-Smirnov: se puede usar para tamaos de muestras grandes y para otras distribuciones aparte de la normal.
  • Anderson-Darling: Modificacin de Kolmogorov-Smirnov para tamaos de muestras pequeas.

Por ahora, slo se hablar de Shapiro-Wilk y Kolmogorov-Smirnov.

Shapiro-Wilk

En el ejemplo de abajo, la funcin rnorm saca n datos aleatorios de una distribucin normal con media = m y desviacin estndar = s. Despus, a esos datos se les aplica la prueba Shapiro-Wilk para probar si tienen una distribucin normal o no.

Recuerda: Generalmente no sabemos la distribucin de los datos a priori. En este ejemplo sabemos que los datos vienen de una distribucin normal entonces esperamos que la prueba Shapiro-Wilk NO rechace h0. Es decir, p-value tiene que ser mayor a 0.05.


n = 30 # num de datos
m = 5  # media
s = 2  # desv estndar

# Datos aleatorios de una distribucin normal
set.seed(123) # para reproducibilidad
norm_data = rnorm(n, m, s)

# prueba de normalidad; h0: norm_data est distribuida de manera normal
shapiro.test(norm_data) 

    Shapiro-Wilk normality test

data:  norm_data
W = 0.97894, p-value = 0.7966

Shapiro-Wilk arroja dos valores:

  • W es el valor estadstico que arroja la prueba despus de aplicar la frmula de Shapiro-Wilk a los datos.
  • p-value toma valores entre 0 y 1. Sirve para cuantificar qu tan confiados estamos de que la distribucin de los datos tienen una distribucin normal. Mientras ms cercano est de 1, los datos se asemejan ms a la normal. Un p-value < .05 nos sirve para definir el umbral en el que podemos decir que los datos son suficientemente diferentes de una distribucin normal para rechazar h0.

Los resultados de shapirto.test se pueden guardar en una variable para acceder a ellos posteriormente.

shapiro = shapiro.test(norm_data)

Ahora probemos a Shapiro-Wilk con datos que sabemos que no vienen de una distribucin normal.

En el ejemplo de abajo generamos datos aleatorios de una distribucin poisson con una lambda = l. En las distribuciones tipo poisson, lambda es como si fuera la media y es el nico parmetro que se necesita.

Como sabemos que los datos NO vienen de una distribucin normal esperamos que shapiro-wilk rechace la h0.

n = 30 # nm de datos
l = 5  # lambda

# datos aleatorios de una distribucin poisson
set.seed(50)
pois_data = rpois(n, l)

shapiro.test(pois_data)

    Shapiro-Wilk normality test

Problemas de Shapiro-Wilk

Como cualquier prueba estadstica, Shapiro-Wilk slo nos dice qu tan probable es que nuestra muestra tenga una distribucin normal. Pero habr ocasiones en que la vida arroje muestras que parecen normales y que Shapiro-Wilk falla en rechazar h0. Esto generalmente ocurre con un tamao de muestra pequea.

En el caso de abajo, los datos vienen de una distribucin poisson y Shapiro-Wilk NO rechaza h0.

set.seed(100)

n = 15 # nmero de datos
lambda = 5 # lambda es como la media para distribuciones poisson
pois_data = rpois(n, lambda)
shapiro.test(pois_data)

    Shapiro-Wilk normality test

data:  pois_data
W = 0.95387, p-value = 0.5873

Otro problema es el tamao de la muestra. Las pruebas de normalidad comparan datos contra distribuciones tericas. Pero en la realidad nada se distribuye perfectamente normal. Si tenemos una muestra muy grande, las pruebas estadsticas tienen mayor poder estadstico y pueden detectar pequeas desviaciones de las distribuciones tericas. sta es la razn por la que no se recomienda que la prueba Shapiro-Wilk se utilice con ms de 5 mil datos. De hecho, la funcin de R para Shapiro-Wilk lanza un error si la usas con ms de 5 mil datos.

Ahora, trata de usar Shapiro-Wilk con una muestra de ms de 5 mil datos

Kolmogorov-Smirnov n > 5 mil

La prueba de Kolmogorov-Smirnov se puede usar para un tamao de muestra grande.

Las desventajas son que:

  • slo sirve para distribuciones contnuas
  • hay que especficar contra qu distribucin queremos compararar

En el ejemplo de abajo se genrea una muestra con tamao mayor a 5 mil y se usa Kolmogorov-Smirnov (ks.test) para probar que tenga una distribucin normal. Observa que el segundo argumento de ks.test es pnorm, que en R hace referencia a la distribucin de probabilidad acumulativa de una curva normal, y le siguen la media y la desviacin estndar de los datos.


n = 5001 # tamao de la muestra
m = 5 # media
s = 2 # desv. estndar
set.seed(5)
datos_masivos = rnorm(n, m , s)
ks.test(datos_masivos, "pnorm", mean(datos_masivos), sd(datos_masivos))

    One-sample Kolmogorov-Smirnov test

data:  datos_masivos
D = 0.01008, p-value = 0.6898
alternative hypothesis: two-sided

Los resultados que arroja ks.test son parecidos a los de Shapiro-Wilk, slo que en lugar de obtener el estadstico W, se obtiene el estadstico D. La p-value se interpreta igual (rechaza h0 con p-value < 0.05 ).

Kolmogorov-Smirnov se puede usar para probar si los datos vienen de otras distribuciones adems de la normal. En el cdigo de abajo se generan datos a partir de una distribucin gamma y se usa Kolomogorov-Smirnov para probar la h0 de que los datos tengan una distribucin gamma. Observa que se tiene que especficar la distribucin acumulativa y los parmetros que le corresponden a la distribucin de comparacin.

n = 5001
shape = 10
scale = 10

set.seed(10)
gamma_data = rgamma(n, shape, scale)

ks.test(gamma_data, "pgamma", shape, scale)

    One-sample Kolmogorov-Smirnov test

data:  gamma_data
D = 0.013391, p-value = 0.3312
alternative hypothesis: two-sided

Pruebas de normalidad grficas: QQ plot

En la seccin anterior vimos que en ocasiones las pruebas de normalidad fallan en detectar que los datos no tienen una distribucin normal. Tambin vimos que si aumentamos la cantidad de datos aumenta su poder estadstico y son capaces de cometer errores tipo 1. Esto ltimo se est volviendo un problema porque cada vez tenemos mayor capacidad de procesar y almacenar datos masivos. Debido a esto es recomendable recurrir a los QQ-plots, o plots cuantil-cuantil.

Los QQ-plots es una herramienta visual para ver si los datos tiene nuna distribucin normal o no. De manera muy simplificada, los qq-plots toman cuantiles de los datos y los grafican en funcin de los cuantiles de una distribucin normal terica. Si los datos siguen una distribucin normal entonces los datos siguen una lnea recta.

Los ejemplos de abajo toman los datos antes generados para ver en los QQ-plots cmo se ajustan a una distribucin normal. Observa cmo los datos generados aleatoriamente de una distribucin normal tienden a seguir la recta. Sin embargo, los datos poisson no y tienen un patrn de escalera caracterstico de una distribucin discreta.

Datos normales

qqnorm(norm_data)
qqline(norm_data, col="red", lw=2)

Datos poisson

La ventaja de los qq-plots es que se pueden usar para ver si los datos siguen otras distribuciones a parte de la normal.

La interpretacin es la misma. Si los datos siguen la lnea recta, los datos vienen de esa distribucin.

Las libreras car o qualityTools tienen funciones qqPlot para poder hacer grficas cuantil-cuantil con otras distribuciones
